> * iphone telegram版本在 5.0.16 的可以用以下链接汉化

繁体中文：[tg://setlanguage?lang=zh-hant-raw](tg://setlanguage?lang=zh-hant-raw)

Android 上Telegram X 和Telegram 最大的不同是不同开发者做的两个产品。
